# Python学生管理系统 python719

本项目是一款基于Python的控制台学生管理系统，旨在帮助学校或教育机构高效管理学生信息。

## 技术栈

- Python 3.x
- 无框架

## 功能模块

- 学生信息管理
  - 添加学生信息
  - 删除学生信息
  - 修改学生信息
  - 查询学生信息
- 成绩管理
  - 录入成绩
  - 修改成绩
  - 查询成绩

## 系统角色

- 管理员：负责整个系统的学生信息和成绩管理

## 运行环境

- 操作系统：兼容Windows、Linux和Mac OS
- Python版本：Python 3.x

## 部署步骤

1. 确保Python 3.x环境已安装
2. 克隆或下载项目源码
3. 在项目目录下运行`main.py`

## 目录结构

```
├── data
│   └── students.json       # 存储学生信息的JSON文件
├── main.py                 # 程序主入口
└── src
    ├── manage.py           # 管理学生信息模块
    └── score.py            # 成绩管理模块
```

## 核心亮点

- 纯Python实现，易于理解和扩展
- 提供控制台界面，方便快捷地进行学生信息管理
- 使用JSON文件存储数据，无需数据库支持，降低部署成本
- 符合教育机构日常管理需求，提高工作效率
- 代码结构清晰，方便二次开发或功能扩展
